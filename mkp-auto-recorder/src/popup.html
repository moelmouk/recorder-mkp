Paramètres

Aide

Enregistrement

  1. Donnez un nom à votre scénario.
  2. Sélectionnez un groupe si nécessaire.
  3. Cliquez sur « Enregistrer », réalisez vos actions, puis « Arrêter ».

Lecture

  • Utilisez « Rejouer » pour exécuter le scénario courant.
  • Dans l’onglet Groupes, cliquez sur le bouton lecture d’un dossier racine pour rejouer tous ses scénarios (y compris les sous-dossiers).
  • Activez « Utiliser le timing réel » pour respecter les délais enregistrés.

Gestion des dossiers et scénarios

  • Créer un dossier : Saisissez un nom et cliquez sur "+"
  • Créer une hiérarchie : Utilisez des "/" pour séparer les niveaux
    Exemple: Projet/Module/Test créera une arborescence complète
  • Bouton "+" : Apparaît à côté de chaque dossier pour ajouter des sous-dossiers
  • Bouton de lecture : Disponible sur chaque dossier pour lancer tous les scénarios du dossier et de ses sous-dossiers
  • Scénarios : Affichés directement dans leur dossier avec des boutons pour les charger ou les éditer
  • Compteur : Affiche le nombre total de scénarios dans chaque dossier (y compris les sous-dossiers)
Astuce : Vous pouvez glisser-déposer les scénarios entre les dossiers

Paramètres (Backup/Restore)

  • Générez un backup JSON de vos données.
  • Copiez ou téléchargez le backup.
  • Restaurez à partir d’un JSON valide.
Astuce: utilisez des dossiers pour organiser vos scénarios par fonctionnalité ou parcours utilisateur.
